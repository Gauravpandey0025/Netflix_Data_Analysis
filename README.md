# Netflix_Data_Analysis
This project focuses on analyzing the popularity of movies across different genres. By cleaning and processing a raw movie dataset, I aimed to uncover meaningful patterns that highlight which genres attract the most audience engagement and interest. The analysis was conducted using structured visualizations, offering clear and intuitive insights.
# 📊 Netflix Data Analysis Project

##  Project Overview

This project involves a comprehensive analysis of Netflix's movie and TV show catalog. The objective is to uncover insights about content distribution by genre, country, ratings, release year, and more through exploratory data analysis and visualization techniques. It also includes essential steps like data cleaning, feature extraction, and pattern identification.

---

## 🛠 Tools & Technologies

- **Language**: Python  
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn  
- **Environment**: Jupyter Notebook / Google Colab  
- **Dataset**: [Netflix Dataset on Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)

##  Analysis Performed

- Total number of movies and TV shows on Netflix  
- Distribution by **type**, **genre**, and **rating**  
- Analysis of **content release trend** over years  
- Country-wise content production and availability  
- Duration analysis for TV shows vs movies  
- Extraction of useful features from the dataset (e.g., year, country, cast)

---

##  Key Insights

- The volume of content has increased significantly post-2015  
- The **United States** produces the highest number of shows and movies  
- **Drama**, **International Movies**, and **Comedies** are the most common genres  
- Most TV Shows have durations in seasons; Movies are generally 90–120 minutes long
